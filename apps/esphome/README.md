# ESPHome

ESP device management and firmware compiler.

## Overview

[ESPHome](https://esphome.io/) compiles firmware for ESP8266/ESP32 devices and provides a dashboard for managing them.

## Repository Structure

```
esphome-k8s/
├── deployment.yaml
├── ingress.yaml
├── service.yaml
└── volume.yaml
```

## Usage

- Manage ESP device configurations
- Compile and upload firmware over-the-air
- MQTT integration with mosquitto
